Obsidian 是一款功能强大的 Markdown 笔记工具，以其灵活的双向链接和丰富的插件生态受到广泛喜爱。熟练掌握 Obsidian 的默认快捷键，能让你的笔记管理、文本编辑和知识整理效率翻倍！本指南将从文本格式、笔记操作、导航管理、视图切换、插件设置等多个维度，带你全面了解 Obsidian 的快捷键体系，助你打造高效工作流！

1️⃣ 文本格式：快速美化笔记内容 ✍️
在 Obsidian 中，格式化文本是笔记整理的基础。以下快捷键帮你轻松实现专业排版：

```
加粗：Ctrl + B（**文本**）
斜体：Ctrl + I（*文本*）
删除线：~~文本~~
高亮：==文本==
行内代码：`代码`
数学公式：Ctrl + /（$公式$ 或 $$块公式$$）
注释（隐藏文本）：%% 内容 %%
切换待办事项：Ctrl + L（- [ ] 或 - [x]）
转换为列表：Ctrl + Shift + L
小贴士：想插入复杂格式（如引用或代码块）？直接输入 Markdown 语法，如 > 引用 或 ````代码块` ```，效率更高！
```
2️⃣ 段落与结构：组织清晰的笔记 📝
良好的笔记结构让信息一目了然。以下是常用结构化编辑方式：

标题：# 一级标题、## 二级标题（最多六级）
无序列表：- 项目
有序列表：1. 项目
任务列表：- [ ] 未完成 或 - [x] 已完成
引用：> 文本 或 > [!tip] 提示
代码块：
```语言
代码内容
AI写代码
markdown
1
2
```

```python
print("Hello, Obsidian!")
1
2
3
```

水平分割线：---（需上下空行）

3️⃣ 基本操作：快速创建与查找 🏗️
高效的笔记管理离不开顺手的基础操作。以下快捷键让你事半功倍：

新建笔记：Ctrl + N
保存笔记：Ctrl + S
在当前文件夹新建子笔记：Ctrl + Shift + N
打开快速切换：Ctrl + O（快速查找文件）
搜索：
笔记内搜索：Ctrl + F
查找下一个/上一个：Ctrl + G / Ctrl + Shift + G
全库搜索：Ctrl + Shift + F
恢复最近关闭的笔记：Ctrl + Shift + T
添加/移除注释：Ctrl + /

4️⃣ 导航与标签：无缝切换笔记 🗂️
Obsidian 支持多标签页和历史导航，快捷键让切换如行云流水：

标签页切换：
下一个：Ctrl + Tab 或 Ctrl + PageDown
上一个：Ctrl + Shift + Tab 或 Ctrl + PageUp
特定标签：Ctrl + 1~9（1~9 为标签序号，9 为最后一个）
标签页管理：
新建：Ctrl + T
在右侧新建：Ctrl + Shift + N
关闭当前：Ctrl + W
关闭所有：Ctrl + Shift + W
导航历史：
返回：Alt + ←
前进：Alt + →
链接导航：
打开链接：Ctrl + Enter
新标签页打开：Ctrl + Alt + Enter
新窗口打开：Ctrl + Alt + Shift + Enter
5️⃣ 面板与视图：灵活调整工作区 🖥️
Obsidian 的多面板和视图切换功能非常强大，快捷键让操作更高效：

打开命令面板：Ctrl + P（快速执行命令）
切换编辑/预览模式：Ctrl + E
分割面板：
不同方向：Ctrl + Shift + 1/2/3
双面板模式：Ctrl + Shift + D
预览窗口滚动：Alt + ↑/↓
查看关系图谱：Ctrl + G
6️⃣ 插入元素：丰富笔记内容 🔗
快速插入链接、标签等元素，让笔记更具互联性：

内部链接：[[笔记名]]（输入 Ctrl + K 快速插入）
标签：#标签名
空锚点：\[[#]]
```
尾注：[[^]]
数学公式块：$$ 公式 $$
文档属性：Ctrl + ;
```
7️⃣ 插件与设置：个性化你的 Obsidian ⚙️
Obsidian 的插件生态让功能无限扩展，以下快捷键助你快速管理：

打开设置：Ctrl + ,
打开插件商店：Ctrl + Shift + P
查看已安装插件：Ctrl + Shift + X
查看帮助：F1
提示：在“设置 → 热键”中自定义快捷键，打造专属工作流！
